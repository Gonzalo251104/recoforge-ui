# Recoforge UI

Frontend application for **Recoforge**, a recommendation system backend built with FastAPI.

This project is intended as a technical playground to demonstrate how a modern React frontend consumes a Python-based recommendation API.

---

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS (v4)
- TanStack React Query
- Axios

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
npm install
```

### Environment variables

Create a .env file in the project root:

```bash
VITE_API_URL=http://localhost:8000
```

Make sure the Recoforge backend is running at that URL.

### Run the project

```bash
npm run dev
```

The app will be available at:

```bash
http://localhost:5173
```
